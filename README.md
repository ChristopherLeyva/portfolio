# Portfolio

Next.js 15 portfolio built with TypeScript, Tailwind CSS v4, and shadcn/ui.

## Quick start

```bash
bun install
bun dev
```

Open [localhost:3000](http://localhost:3000).

## Commands

| Command          | Description            |
| ---------------- | ---------------------- |
| `bun dev`        | Development server     |
| `bun build`      | Production build       |
| `bun start`      | Production server      |
| `bun test:run`   | Run tests              |
| `bun lint`       | Lint code              |
| `docker compose up` | Full stack with Docker |

## Stack

- **Framework:** Next.js 15
- **Runtime:** Bun
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Testing:** Vitest
- **Hosting:** Vercel
